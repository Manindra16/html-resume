Company: Innominds Software SEZ

Title: Object Detection Model Training

Duration: 26 Oct 2023 to 26 Dec 2023.

Overview:

  • Objection detection model training using SSD Mobile Net.
  • MobilenetSSD is an object detection model that computes the bounding box and category of an object from an input image.
  • The Single Shot Detector (SSD) object detection model uses Mobilenet as its backbone and can achieve fast object detection optimized for mobile devices.
